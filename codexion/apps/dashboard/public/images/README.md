Este diretório contém as imagens locais do projeto.

Para baixar todas as imagens externas usadas no site, execute:

```bash
bash scripts/download-assets.sh
```

Isso fará download dos arquivos para:

- public/images/hero
- public/images/showcase
- public/images/progress

Após o download, atualize os caminhos das imagens para usar /images/... (se ainda estiverem usando URLs externas).
