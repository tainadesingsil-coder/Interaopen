# Projetos Separados

Esta pasta foi criada para manter os ambientes isolados:

- `codexion-dashboard/`: frontend premium da plataforma Codexion (Next.js + Tailwind + Framer Motion + Recharts + UI estilo Shadcn).
- `solarenergy-original/`: snapshot do projeto original antes da adaptação visual para Codexion.

## Como rodar o Codexion localmente

```bash
cd projetos-separados/codexion-dashboard
npm install
npm run dev
```

Depois, abra: `http://localhost:3000`
